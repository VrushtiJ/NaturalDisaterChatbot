Install node js: https://nodejs.org/en/download/

Install expo cli: npm install -g expo-cli

Initialize a repository:
expo init AwesomeProject
cd AwesomeProject

Install react native chat UI library: https://github.com/FaridSafi/react-native-gifted-chat
npm install react-native-gifted-chat --save

Run the app:
npm start

To run the app on your phone, install expo app and then scan the QR code which appears on the broswer after running npm start.
